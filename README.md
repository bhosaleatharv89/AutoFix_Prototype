# 🚗 Doorstep Vehicle Service System

A web-based system that demonstrates the workflow of a doorstep vehicle service platform. It allows customers to book services and mechanics to manage requests, simulating a real-world service lifecycle.

---

## 📌 Features

### 👤 Customer Interface

* Select vehicle service (oil change, puncture repair, etc.)
* View nearby mechanics
* Book service with date and time
* Booking confirmation display
* Service status tracking (Requested → Completed)

---

### 🔧 Mechanic Interface

* View incoming service requests
* Accept or reject requests
* Update job status (On the Way, Working, Completed)
* Generate service bill and view summary

---

## 🔄 System Workflow

Customer → Select Service → Choose Mechanic → Book Appointment →
Mechanic → Accept Request → Travel → Service → Completion → Billing

---
